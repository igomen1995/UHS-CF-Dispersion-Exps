# FOLDER CONTENT
This folder contains the raw data acquired for Dispersion Core Flooding Experiments using the experimental setup described in ...., which includes:
- Bronkhorst Mass Flow Meter data (temperature and density)

# Label for each experiment data
The experiments were run at different conditions, using different gases and are described as follow:

CF-H2-CO2-T32-P1500-H-Q1-NS-R1
where:
- CF is type of experiment: Core flooding through core (CF) or through bypass lines (BP)
- H2 displacing fluid
- CO2 displaced fluid
- T32 temperature in °C
- P1500 pressure in psi
- H position of the core horizontal (H) or vertical (V)
- Q1 injection flow rate in ml/min
- NS no sampling for GSMS or SS sampling for GCMS. It implies lack of data in the portable gas detectors
- R1 running number. The experiment (with same characteristics) can be repeated more than once
